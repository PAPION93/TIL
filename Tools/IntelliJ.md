### 1. 메인 메소드와 System.out.println()
- `psvm`
- `sout`

### 2. 선언부 바로 보기
- `Opt + Space`
- `Shift + Ctrl + I`

### 3. 코드 정렬
- `Cmd + Opt + L`
- `Ctrl + Alt + L`

### 4. 전체 프로젝트에서 문자열 검색(교체)
- `Cmd + Shift + F(R)`
- `Ctrl + Shift + F(R)`
